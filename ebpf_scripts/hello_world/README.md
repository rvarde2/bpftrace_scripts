# Kernel-Space Program
## Hook
    SEC("tp/syscalls/sys_enter_execve")
    - tp: tracepoint
    - syscalls/sys_enter_execve: event
## Callback
    int handle_tp(void *ctx)
    - ctx: contains arguments passed to the syscall

# User-Space Program
## ByteCode
    hello.skel.h file will be generated by bpftool
## Send ByteCode to kernel
    hello_bpf__open_and_load
## Plug and Run the code
    hello_bpf__attach(skel)

# RUN
## Terminal 1
    sudo ./hello

## Terminal 2
    sudo cat /sys/kernel/debug/tracing/trace_pipe
